resultados para "transgénico"
 transgénico (cat. nominal) - planta usada para consumo humano cujo material genético foi alterado através de técnicas de engenharia genética, combinando numa só molecula moléculas de ADN de diferentes fontes, de modo a criar um novo gene que é implantado nesse organismo para obter características específicas

 transgénico (cat. adjectival) - cujo material genético foi alterado através de técnicas de engenharia genética, combinando, in vitro, moléculas de ADN de diferentes fontes numa só molécula, de forma a criar um novo gene que é implantado num organismo para que este desenvolva características específicas
