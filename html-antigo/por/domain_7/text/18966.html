O protocolo ATM acrescenta no envio mais alguns bits fundamentais aos pacotes de dados cortados em pequenos segmentos de tamanho fixo.

Asynchronous Transfer Mode, ou simplesmente ATM é um protocolo de redes de computadores para comunicação de alto débito, que transporta os dados em pacotes de tamanho fixo (53 bytes; 48 bytes de dados úteis e 5 Bytes de controlo ). Este método de transmissão de dados envia de forma cíclica e contínua os pacotes de tamanho fixo contendo segmentos de informação que é reconstruída na recepção.
