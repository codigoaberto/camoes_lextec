PPP, protocolo PPP, protocolo ponto a ponto
definição | textos ilustrativos | rede conceptual

Em termos do equipamento utilizado, o acesso via ligação PPP é idêntico ao tipo de acesso limitado a caracteres (...). No entanto, uma ligação PPP é tecnicamente superior a uma ligação em modo texto, visto que nem tudo o que passa pela linha é considerado texto. A grande diferença reside no protocolo (...). Na base das comunicações continuam a estar os muito falados bits (ou zeros e uns). Porém, o protocolo PPP interpreta-os e organiza-os de maneira diferente, de modo a permitir que para um programa empregando gráficos (Netscape por exemplo), os bits signifiquem pontos e cores de uma imagem, e para um programa puramente textual (como o Telnet) esses mesmos bits signifiquem caracteres.

(http://diferencial.ist.utl.pt/edicao/19/ppp.htm)

O protocolo ponto-a-ponto (PPP - "Point to Point Protocol") é um método standard para estabelecer uma ligação de rede entre dispositivos ligados em rede. Existem diferentes formas de PPP, como PPPoE e PPPoA, que envolvem um processo de autenticação e que requerem um nome de utilizador e uma password para autorizar o acesso à rede. Este é o tipo de ligação recomendado para a maioria dos utilizadores.
