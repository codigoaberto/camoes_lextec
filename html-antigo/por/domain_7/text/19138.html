Tal como existem unidades para medir pesos (miligrama, grama, kilograma,...) e distâncias (milímetro, centímetro, decímetro, metro, ...), também existem unidades para medir informação (bit, byte, kilobyte, megabyte, gigabyte, terabyte, ...).

Um bit é a unidade de informação correspondente a um único dígito binário. Um byte são 8 bits. Um kilobyte são 210 = 1024 bytes. Um megabyte são 220 kilobytes (aproximadamente 1 milhão de bytes). Um gigabyte são 230 bytes (...). Um terabyte são 240 bytes (...).
