modem
definição | textos ilustrativos | rede conceptual

Um modem é uma caixa pequena ou uma placa de expansão que liga o PC à linha telefónica. (...) O termo "modem" é a abreviatura para modulator/demodulator, que é uma descrição técnica daquilo que o aparelho faz. Para funcionar com a rede telefónica, para a qual é essencial um sistema sonoro analógico, os dados do computador têm de ser convertidos para um sinal compatível, e é nessa altura que entra o modem.
