servidor proxy, servidor de proxy
definição | textos ilustrativos | rede conceptual

Na UA, para aceder à internet, é necessário indicar ao equipamento que se está a utilizar um servidor PROXY. O servidor proxy serve como concentrador dos pedidos de acesso à internet. Nos equipamentos móveis, para se aceder à internet por intermédio de um servidor Proxy, configura-se a rede wireless para usar uma rede de trabalho.

(http://www.ua.pt/ReadObject.aspx?obj=2098)

Normalmente um servidor com cache passiva, quando coloca um documento em cache, associa-lhe um tempo de vida. Enquanto este tempo não expira, todos os pedidos são atendidos com o documento em cache. Quando expirar o tempo de vida, e havendo um pedido para um documento, o Servidor de Proxy solicita um novo pedido ao host destino, actualizando a cache. Se o espaço em disco da área de cache encher, o Servidor remove os documentos mais antigos, de forma a poder armazenar novos documentos.
