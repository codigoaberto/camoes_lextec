Wi-Fi, rede Wi-Fi
definição | textos ilustrativos | rede conceptual

Para utilizar o WLAN precisa de um dispositivo que suporte esta tecnologia e também terá de se encontrar numa área com uma rede Wi-Fi activa (denominada um "hot spot"). Conforme as definições de segurança da rede, poderá ter de introduzir um código para se ligar.

Wi-Fi (...) refere-se a uma gama específica de redes WLAN, nomeadamente a IEEE802.11, que actualmente são as WLAN mais vulgarmente utilizadas.
