Os diferentes tipos de interfaces de voz são os seguintes:

Interfaces analógicos, [que] permitem a ligação de terminais telefónicos analógicos com marcação decádica ou multifrequência.

Interfaces digitais, [que] permitem a ligação de terminais telefónicos digitais (...).

Interfaces IP, [que] através do gateway VoIP permitem a ligação de terminais telefónicos IP (...) ligados a rede LAN ou terminais IP compatíveis com o protocolo IP H323. (...)

Interfaces RDIS S0, [que] permitem a ligação de terminais telefónicos RDIS.

Interfaces DECT, [que] permitem antenas de 2 ou 4 canais rádio para comunicação simultânea.

(http://www.tecnicontrol.pt/telecom/prod_centrais_6501_RL.html)

As redes de telecomunicações actuais suportam dois modos de marcação, marcação por disco rotativo ou por tons.
