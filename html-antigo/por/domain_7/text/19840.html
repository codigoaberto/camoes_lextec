O protocolo ponto-a-ponto (PPP - "Point to Point Protocol") é um método standard para estabelecer uma ligação de rede entre dispositivos ligados em rede. Existem diferentes formas de PPP, como PPPoE e PPPoA, que envolvem um processo de autenticação e que requerem um nome de utilizador e uma password para autorizar o acesso à rede. Este é o tipo de ligação recomendado para a maioria dos utilizadores.
