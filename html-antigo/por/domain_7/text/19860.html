PUK, código PUK, código pessoal de desbloqueio
definição | textos ilustrativos | rede conceptual

No caso de se introduzir o PIN errado três vezes, é necessário recorrer a um outro código, o PUK, para se poder utilizar o telemóvel.

(http://www.telemoveis.com/articles/item.asp?ID=45)

Se introduzir um código PIN incorrecto várias vezes, o telemóvel poderá pedir-lhe um código PUK (Personal Unblocking Key - Código Pessoal de Desbloqueio). O código PUK é um código de 8 dígitos fornecido pelo seu operador de rede ou fornecedor de serviços.

(http://www.nokia.pt/A4269078)
