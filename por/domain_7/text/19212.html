rede local sem fios, WLAN, rede WLAN
definição | textos ilustrativos | rede conceptual

Uma rede local sem fios (WLAN) é um sistema flexível de comunicações de dados, implementado como uma extensão ou uma alternativa para uma rede local com fios no interior de um edifício. Utilizando ondas electromagnéticas, as WLANs transmitem e recebem dados através do ar, diminuindo a necessidade de ligações com fios. Assim, as WLANs combinam a conectividade de dados com a mobilidade do utilizador e, através de uma configuração simplificada, permitem a criação de redes locais (LANs) móveis.

(http://www.pmelink.pt/pmelink_public/EC/0,1655,1005_43337-3_41103--View_429,00.html)

Para utilizar a WLAN precisa de um dispositivo que suporte esta tecnologia e também terá de se encontrar numa área com uma rede Wi-Fi activa (denominada um "hot spot"). Conforme as definições de segurança da rede, poderá ter de introduzir um código para se ligar.

Wi-Fi (...) refere-se a uma gama específica de redes WLAN, nomeadamente a IEEE802.11, que actualmente são as WLAN mais vulgarmente utilizadas.
