interface de utilizador, software de interface com o utilizador
definição | textos ilustrativos | rede conceptual

A interface de utilizador pode ser utilizada para enviar ordens a um dispositivo ou electrodoméstico da casa, bem como para supervisionar o ambiente doméstico. (...) As ordens podem ser verbais, tácteis ou outras, consoante a necessidade. Este aspecto do sistema doméstico é particularmente útil para indivíduos deficientes, na medida em que se pode traduzir numa melhoria da respectiva qualidade de vida, dotando-os de maior independência. Por exemplo, uma interface de utilizador baseada em ordens verbais seria extremamente útil para indivíduos com limitações motoras.

(http://www.domoticapt.com/ligacaosistemadomotico.php)

O software de interface com o utilizador permite configurar todos os parâmetros necessários ao funcionamento do sistema. Em qualquer altura pode alterar a configuração. A mesma entra em funcionamento de imediato.

(http://www.primetec.pt/files/primesat_controller_manual_pt.pdf)
