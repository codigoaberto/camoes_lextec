rede broadcast, rede de broadcast, rede multiponto
definição | textos ilustrativos | rede conceptual

Numa rede LAN ("Local Area Network") é possivel um controlo físico sobre as linhas mais efectivo, contudo é necessário atender a um factor adicional: muitas redes locais são ainda do tipo "broadcast".

Numa rede broadcast, toda a informação chega a todos os nós (...). Este tipo de rede causa grandes problemas de confidencialidade pois em qualquer ponto é possivel colocar um dispositivo de escuta e receber informação trocada entre quaisquer dois nós.

(http://www.dei.isep.ipp.pt/~andre/documentos/seguranca-fisica.html)

As implementações mais correntes de redes locais não utilizam comutação, existe um meio de transmissão que é partilhado por todos os nós. Quando um nó emite dados, todos os outros nós recebem esses dados. Cabe a cada nó analisar o endereço de destino para determinar se os dados lhe são destinados.

Pelo facto de todos os nós receberem os dados, este tipo de redes são conhecidas por redes de broadcast ou multiponto.

(http://www.dei.isep.ipp.pt/~andre/documentos/mac.html)

(...) As redes multiponto partilham entre todos os dispositivos de rede um meio de transmissão único, através do qual toda a informação é transmitida. A identificação do destinatário é feita através de uma identificação unívoca, ou endereço de rede, embebida na informação transmitida. Quando os dispositivos de rede detectam a presença de informação na rede, só aquele que se identificar com o endereço transmitido é que o vai processar, enquanto que os outros o ignoram.

(http://lsm.dei.uc.pt/ib/comunicacoes/redes/classificacao.html)
