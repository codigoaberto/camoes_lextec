HTML
definição | textos ilustrativos | rede conceptual

A Web surgiu associada a uma nova linguagem de composição de documentos, o HTML (HyperText Markup Language), e a um novo protocolo, o HTTP (HyperText Transfer Protocol). O HTML é uma linguagem destinada à criação de documentos em hipertexto, que funciona com base em certas palavras e símbolos - chamados links ou hyperlinks - que remetem para novas páginas de texto ou secções do mesmo documento. No caso do HTML, o sistema de hipertexto permite passar não só de um ponto para o outro no mesmo documento, como de um documento para outro, podendo estes estar situados em qualquer parte da [rede] mundial (WWW) de servidores que compõem este sistema.
